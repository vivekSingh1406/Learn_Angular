## 🔹 1. What is an API?

**API (Application Programming Interface)** is a way for two applications to communicate with each other.

In Angular:
- **Frontend (Angular App)** → sends request  
- **Backend (Server)** → processes request  
- **API** → acts as a bridge between them  

### ✅ Simple Analogy:
- You → Client (Angular App)
- Waiter → API
- Kitchen → Server

You place an order → API sends it → Server processes → Response comes back.

---

## 🔹 2. Why Do We Use APIs in Angular?

Angular is a frontend framework, so it does not directly interact with databases.

👉 APIs help Angular:
- Fetch data from server
- Send user input to server
- Update existing data
- Delete records

---

## 🔹 3. What is a REST API?

**REST (Representational State Transfer)** is a standard way of designing APIs.

### Key Features:
- Uses HTTP methods (GET, POST, etc.)
- Works with URLs (endpoints)
- Returns data in JSON format
